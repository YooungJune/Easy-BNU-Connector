# Easy-BNU-Connector 0.0.0

连接北京师范大学校园网的Python程序

所需模块：Requests、BeautifulSoup4

methods.py 是由 methods.js 通过 Js2Py 转换成的，methods.js 取自认证网关

程序运行不需有 methods.js


感谢 YooungJune 的测试
